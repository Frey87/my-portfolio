# Introduction to HTML, CSS, & JavaScript Portfolio

This repository contains my final portfolio project completed as part of the **IBM Skills Network - Introduction to HTML, CSS, & JavaScript** course.

## Overview

The project demonstrates the fundamentals of front-end web development by creating a responsive single-page portfolio using HTML, CSS, and JavaScript.

## Features

- Personalized Home section
- About Me section
- Skills showcase
- Project portfolio
- Recommendations section
- Interactive recommendation form
- JavaScript popup confirmation
- Scroll-to-top button

## Technologies Used

- HTML5
- CSS3
- JavaScript

```
.

```

## Learning Outcomes

During this project I practiced:

- HTML document structure
- CSS styling and layout
- Navigation using anchor links
- DOM manipulation
- JavaScript functions
- Event handling
- Dynamic content creation
- Responsive web design fundamentals
